# AI Frontend Kit

Este diretório define como agentes AI devem gerar e revisar o frontend Angular.

## Ordem recomendada de uso
1. Ler `rules/`.
2. Executar prompt base em `prompts/`.
3. Validar com `checklists/`.
4. Consultar padrão em `examples/`.

## Integração com agentes
- Os agentes em `.github/agents` devem referenciar este diretório como fonte principal de padrões.
