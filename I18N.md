# I18n — Internacionalização

Sistema de internacionalização runtime implementado para o Product Template Angular.

---

## 📚 Visão Geral

- **Locales suportados:** PT-BR (padrão), EN-US
- **Runtime i18n:** Troca de idioma sem rebuild
- **Auto-detecção:** Idioma do navegador
- **Persistência:** localStorage
- **Interpolação:** Variáveis dinâmicas
- **Type-safe:** TypeScript

---

## 🚀 Uso Rápido

### No Template

```html
<!-- Tradução simples -->
<button>{{ 'common.save' | translate }}</button>

<!-- Com parâmetros -->
<p>{{ 'errors.minLength' | translate:{ min: 3 } }}</p>

<!-- Em atributos -->
<input [placeholder]="'auth.email' | translate" />
```

### No Component

```typescript
import { Component, inject } from '@angular/core';
import { I18nService } from '@/app/core/i18n/i18n.service';
import { TranslatePipe } from '@/app/core/i18n/translate.pipe';

@Component({
  selector: 'app-my-component',
  standalone: true,
  imports: [TranslatePipe],
  template: `<h1>{{ 'products.title' | translate }}</h1>`
})
export class MyComponent {
  readonly i18n = inject(I18nService);
}
```

### Mudar Idioma

```typescript
// Component
i18n.setLocale('en-US');

// Template - Seletor já incluído no shell
<app-language-selector />
```

## 📂 Estrutura

```
src/
├── i18n/
│   ├── pt-BR.json              # Traduções português
│   ├── en-US.json              # Traduções inglês
│   └── i18n.config.ts          # Configuração
└── app/
    ├── core/
    │   └── i18n/
    │       ├── i18n.service.ts       # Service principal
    │       ├── i18n.service.spec.ts  # Testes
    │       ├── translate.pipe.ts     # Pipe {{ | translate }}
    │       └── i18n.interceptor.ts   # Header Accept-Language
    └── shared/
        └── components/
            └── language-selector.component.ts  # Seletor visual
```

## 📝 Adicionar Novas Traduções

1. **Adicionar em PT-BR:**

```json
// src/i18n/pt-BR.json
{
  "myFeature": {
    "title": "Minha Feature",
    "action": "Executar Ação"
  }
}
```

2. **Adicionar em EN-US:**

```json
// src/i18n/en-US.json
{
  "myFeature": {
    "title": "My Feature",
    "action": "Execute Action"
  }
}
```

3. **Usar no template:**

```html
<h1>{{ 'myFeature.title' | translate }}</h1>
<button>{{ 'myFeature.action' | translate }}</button>
```

## 🌐 Namespaces Disponíveis

| Namespace | Uso | Exemplo |
|-----------|-----|---------|
| `common.*` | Ações comuns | `common.save`, `common.cancel` |
| `auth.*` | Autenticação | `auth.login`, `auth.logout` |
| `errors.*` | Erros de validação | `errors.required`, `errors.email` |
| `validation.*` | Mensagens de validação | `validation.emailInvalid` |
| `navigation.*` | Menu e navegação | `navigation.home`, `navigation.products` |
| `{feature}.*` | Feature específica | `products.title`, `users.created` |
